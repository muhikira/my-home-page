My HTML Cheat Sheet

Heading

Headings from 'h1' to '6' identify blocks and sections.Every page must have one 'h1'

Paragraphs

Paragraphs of text go in 'p' tags.

Anchors(i.e links)

Anchor tags create links to other pages. The URL goes in the 'href'attributes. a href="https://duckduckgo.com"> go to DuckDuckgo to search for more information.

Setting 'target="_blank"' will open the link ina new window.See MDN for more details.

UnorderedList

Unordered List are marked with bullet points qnd andin 'ul' tags.Each list item goes in an 'li' tag.

Types of lists:

  • Unordered List('ul')
  • Ordered lists ('ol')
